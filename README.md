# Hermes Voice Telemetry

**Voice-telemetry service (FastAPI + circuit breaker + metrics) para hermes-agent**

## Install

```bash
pip install hermes-voice-telemetry
hermes_voice_telemetry_bootstrap  # se disponível
```

## Compatibilidade

- Python 3.11+
- hermes-agent >= 0.16.0
- Ubuntu 22.04+
